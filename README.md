# Daniel Fortune Ofu-Owoicho — Frontend Developer Portfolio

A modern, responsive portfolio website built with **React 19**, **Vite**, and **TypeScript** (in projects). Showcasing my skills, projects, and experience as a Frontend Developer specializing in React, Next.js, and JavaScript.

Live Demo: [daniel-fortune-portfolio.vercel.app](https://daniel-fortune-portfolio.vercel.app)

## 🚀 Features

- **Modern React Architecture**: Built with React 19 and Vite for lightning-fast development and builds
- **Dark/Light Theme**: Automatic theme detection with localStorage persistence
- **Responsive Design**: Optimized for desktop, tablet, and mobile
- **Smooth Animations**: Polished UI with cinematic scroll experiences inspired by projects
- **Project Showcase**: Detailed case studies of key projects
- **CV Integration**: Includes downloadable CV and project links

## 🛠 Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: CSS3, Responsive Design
- **Build Tool**: Vite
- **Type Checking**: TypeScript (in select projects)
- **Deployment**: Vercel
